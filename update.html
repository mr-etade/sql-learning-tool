UPDATE Command

Modify existing records in a database table

Basic UPDATE

Change a student's grade:

UPDATE Students SET Grade = 'A' WHERE StudentID = 102;

Multiple Fields

Update a student's age and grade:

UPDATE Students SET Age = 21, Grade = 'B' WHERE StudentID = 104;

Multiple Records

Update all computer science courses:

UPDATE Courses SET Department = 'Computer Science Dept' WHERE Department = 'Computer Science';
