INSERT Command

Add new records to a database table

Basic INSERT

Add a new student with all fields:

INSERT INTO Students (StudentID, FirstName, LastName, Age, Grade) VALUES (105, 'James', 'Wilson', 19, 'B');

Partial INSERT

Add a new student with only required fields:

INSERT INTO Students (StudentID, FirstName, LastName) VALUES (106, 'Emma', 'Davis');

Multiple INSERT

Add multiple students at once:

INSERT INTO Students (StudentID, FirstName, LastName, Grade) VALUES (107, 'Tom', 'Lee', 'A'), (108, 'Lisa', 'Chen', 'B');
