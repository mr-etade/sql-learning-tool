COUNT Function

Count the number of rows matching specified criteria

Basic COUNT

Count all students:

SELECT COUNT(*) FROM Students;

COUNT with WHERE

Count students with grade A:

SELECT COUNT(*) FROM Students WHERE Grade = 'A';
