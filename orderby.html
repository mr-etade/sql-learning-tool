ORDER BY Clause

Sort query results in ascending or descending order

Basic ORDER BY

Sort students by last name alphabetically:

SELECT FirstName, LastName FROM Students ORDER BY LastName;

Descending Order

Sort students by age (oldest first):

SELECT * FROM Students ORDER BY Age DESC;

Multiple Columns

Sort students by grade (highest first), then by last name:

SELECT FirstName, LastName, Grade FROM Students ORDER BY Grade ASC, LastName;
