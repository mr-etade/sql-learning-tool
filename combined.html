Combined SQL Clauses

Use multiple clauses together in a single query

WHERE and ORDER BY

Find A students, sorted by last name:

SELECT FirstName, LastName FROM Students WHERE Grade = 'A' ORDER BY LastName;

Multiple Conditions with Sorting

Find courses with 3+ credits, sorted by name:

SELECT CourseName, Credits FROM Courses WHERE Credits >= 3 ORDER BY CourseName;

Complex Combined Query

Find students under 21, sorted by age (oldest first), then by last name:

SELECT * FROM Students WHERE Age < 21 ORDER BY Age DESC, LastName;
